# 要件定義書
## ConversationMovie

**文書バージョン**: 1.0.0  
**作成日**: 2026-05-19

**更新メモ（2026-05-23）**: アップロード入力は「音声ファイル」および「トランスクリプトファイル（`.txt` / `.md` / `.srt` / `.vtt`）」の双方を対象とする。

---

## 1. システム要件概要

### 1.1 コンテンツ処理パイプライン
```
音声入力 → 文字起こし → 話者分離 → AI分析 → 字幕生成 → 動画生成 → YouTube投稿
```

### 1.2 主要ユースケース

#### UC-001: 音声またはトランスクリプトをアップロードして動画を生成する
```
アクター: ユーザー
前提条件: ユーザーがログイン済み
主フロー:
  1. ユーザーが音声ファイルまたはトランスクリプトファイルをアップロード
  2. （音声入力時）システムがWhisperで文字起こしを実行
  3. （音声入力時）システムがpyannoteで話者分離を実行
  4. システムがGeminiで要約・テーマ・感情分析を実行
  5. システムが字幕ファイルを生成
  6. システムがアバター動画を生成
  7. ユーザーが動画プレビューを確認
  8. ユーザーがYouTube投稿を承認
  9. システムがYouTubeへ自動投稿
代替フロー:
  3a. （音声入力時）話者分離失敗 → 単一話者として続行
  9a. YouTube投稿キャンセル → 動画ファイルをダウンロード提供
```

#### UC-002: Shorts動画を生成する
```
アクター: ユーザー
前提条件: 動画生成完了済み
主フロー:
  1. システムが名言・ハイライトを自動抽出
  2. ユーザーがShorts候補から選択（または自動選択）
  3. システムが縦型60秒Shorts動画を生成
  4. ユーザーがYouTube Shortsへ投稿
```

#### UC-003: 処理状況をリアルタイム確認する
```
アクター: ユーザー
主フロー:
  1. ユーザーが処理ステータスページを開く
  2. システムがWebSocket/SSEでリアルタイム進捗を配信
  3. 各ステップの完了通知を表示
```

## 3. 非機能要件詳細

### 3.1 処理パフォーマンス

| 処理ステップ | 入力 | 目標時間 |
|---|---|---|
| ファイルアップロード | 100MB | 30秒以内 |
| 文字起こし | 1分音声 | 30秒以内 |
| 話者分離 | 1分音声 | 60秒以内 |
| Gemini分析 | 10,000トークン | 10秒以内 |
| 字幕生成 | 1,000行 | 5秒以内 |
| 動画生成 | 10分動画 | 10分以内 |

### 3.2 データ保持

| データ種別 | 保持期間 |
|---|---|
| 音声ファイル（元データ） | 30日間（Supabase Storage） |
| 文字起こしテキスト | 無制限（DB） |
| 生成動画ファイル | 7日間（ローカル/Storage） |
| 処理ログ | 90日間 |

### 3.3 エラーハンドリング

- 各処理ステップで失敗した場合、エラー詳細をDBに記録
- ユーザーにエラー内容を通知（UI + メール）
- 部分的失敗の場合は次ステップへの継続可否を判定
- Celeryの自動リトライ（最大3回、指数バックオフ）

## 4. 外部サービス依存関係

| サービス | 用途 | 無料枠 | 代替 |
|---|---|---|---|
| Supabase | DB・Auth・Storage | 500MB DB, 1GB Storage | PlanetScale |
| Gemini API | テキスト生成・分析 | 15 RPM, 1M TPM | OpenAI GPT-4o-mini |
| YouTube Data API | 動画投稿 | 10,000 units/日 | なし（必須） |
| Redis | Celeryブローカー | ローカル実行 | - |

## 5. データフロー

```
[ユーザー]
    ↓ 音声ファイルアップロード
[FastAPI (音声受付)]
    ↓ Supabase Storageに保存
    ↓ Celeryタスクをキュー投入
[Celery Worker - 文字起こし]
    ↓ Whisper実行
    ↓ タイムスタンプ付きテキスト生成
[Celery Worker - 話者分離]
    ↓ pyannote実行
    ↓ 話者ラベルマージ
[Celery Worker - AI分析]
    ↓ Gemini API呼び出し
    ↓ 要約・テーマ・感情・名言・アバターセリフ生成
[Celery Worker - 動画生成]
    ↓ Remotionでアバター動画レンダリング
    ↓ FFmpegで字幕合成
    ↓ Supabase Storageに保存
[Celery Worker - YouTube投稿]
    ↓ YouTube Data API呼び出し
    ↓ 投稿URL記録
[FastAPI (結果返却)]
    ↓ WebSocket/SSEで進捗通知
[ユーザー]
```
