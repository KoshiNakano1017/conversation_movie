## 2. Phase 0: インフラ・基盤構築（2週間）

**目標**: 開発環境が整い、Hello Worldが動く状態

### Week 1
```
□ プロジェクト初期設定
  ├── Gitリポジトリ作成
  ├── ディレクトリ構成の作成
  ├── Docker Compose設定
  └── .env.example 作成

□ バックエンド基盤
  ├── FastAPI + Uvicorn セットアップ
  ├── SQLAlchemy + Alembic 設定
  ├── Supabase接続設定（Auth/DB/Storage）
  └── /health エンドポイント実装
```

### Week 2
```
□ 認証基盤
  ├── Supabase Auth + Google OAuth設定
  ├── JWT検証ミドルウェア
  └── ログイン/ログアウトUI

□ Celery基盤
  ├── Redis セットアップ
  ├── Celeryワーカー設定
  ├── Flowerダッシュボード設定
  └── サンプルタスク動作確認

□ フロントエンド基盤
  ├── Jinja2テンプレート設定
  ├── Tailwind CSS設定
  └── ベーステンプレート作成

✅ マイルストーン: docker compose up → ログイン画面が表示される
```

## 3. Phase 1: MVPコア機能実装（4週間）

**目標**: 音声/トランスクリプト→YouTube投稿の一気通貫フローが動く

### Week 3: 入力処理（音声/トランスクリプト）
```
□ 入力アップロード機能
  ├── アップロードAPI実装
  ├── Supabase Storageへの保存
  ├── ジョブ作成とDB記録
  └── アップロードUI（ドラッグ&ドロップ）

□ Whisper文字起こし
  ├── Whisper mediumモデルの統合
  ├── タイムスタンプ付き文字起こし
  ├── Celeryタスクとして実装
  └── 結果のDB保存

✅ マイルストーン: 音声またはトランスクリプトをアップロード → 分析入力テキストがDBに保存される
```

### Week 4: AI分析
```
□ Gemini API連携
  ├── Geminiクライアント設定
  ├── 要約・テーマ・名言プロンプト
  ├── アバターセリフ生成プロンプト
  ├── YouTubeメタデータ生成
  └── 結果のDB保存

□ パイプライン制御
  ├── Celery chainで処理を連結
  ├── 各ステップのステータス管理
  └── エラー時の記録・通知

✅ マイルストーン: 文字起こし → AI分析結果がDBに保存される
```

### Week 5: 動画生成
```
□ 字幕生成
  ├── SRTファイル生成ロジック
  └── DB保存・ファイル出力

□ Remotionセットアップ
  ├── Remotionプロジェクト作成
  ├── AvatarVideoコンポジション
  ├── アバターコンポーネント（静止画版）
  ├── 字幕オーバーレイコンポーネント
  └── Remotion CLIレンダリング

□ FFmpeg統合
  ├── 字幕合成
  ├── 動画品質設定
  └── 出力ファイル管理

✅ マイルストーン: 処理完了 → MP4動画ファイルが生成される
```

### Week 6: YouTube投稿 + UI完成
```
□ YouTube投稿機能
  ├── YouTube OAuth連携
  ├── 動画アップロードAPI実装
  ├── メタデータ設定
  └── 投稿URL記録

□ リアルタイム進捗UI
  ├── WebSocket実装
  ├── 進捗ステップUI
  └── 完了通知

□ UIブラッシュアップ
  ├── ダッシュボード（ジョブ一覧）
  ├── 動画プレビュー画面
  ├── YouTube投稿設定画面
  └── 履歴ページ

✅ マイルストーン: 音声/トランスクリプトアップロード → YouTubeに自動投稿される（MVP完成）
```

## 8. 進捗管理方法

### 個人開発での推奨ツール
- **GitHub Issues**: タスク管理（このドキュメントのタスクをIssue化）
- **GitHub Projects**: カンバンボード
- **週次レビュー**: 毎週日曜に進捗確認・翌週計画

### KPI（週次確認）
- コミット数: 5件/週以上
- タスク消化率: 計画の80%以上
- 技術的ブロッカーの解決: 48時間以内
