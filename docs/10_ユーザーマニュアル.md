# ユーザーマニュアル
## ConversationMovie v1.0

**作成日**: 2026-05-19

**更新メモ（2026-05-23）**: ユーザー入力は音声ファイルに加え、トランスクリプトファイル（`.txt` / `.md` / `.srt` / `.vtt`）アップロードにも対応。

---

## はじめに

ConversationMovieは、Google Meetなどの会議録音をアップロードするだけで、AIアバターが解説する動画を自動生成してYouTubeに投稿してくれるサービスです。

---

## 1. 初回セットアップ

### 1.1 必要なもの
- Googleアカウント
- YouTubeチャンネル（動画を投稿する場合）
- 入力ファイル（次のいずれか）
  - 音声: mp3, wav, mp4, m4a, ogg, webm
  - トランスクリプト: txt, md, srt, vtt

### 1.2 ログイン方法

1. ブラウザで `http://localhost:8000` にアクセスします
2. **「Googleでログイン」** ボタンをクリックします
3. Googleアカウントを選択してログインします
4. ダッシュボードが表示されれば完了です

### 1.3 YouTubeチャンネルの連携

動画をYouTubeに自動投稿するには、最初に連携設定が必要です。

1. 右上のメニューから **「設定」** を開きます
2. **「YouTubeチャンネルを連携する」** ボタンをクリックします
3. Googleアカウントを選択します
4. YouTubeへのアクセスを **「許可」** します
5. 「連携完了」が表示されれば設定完了です

---

## 2. 入力ファイルのアップロード

### 2.1 アップロード手順

1. ダッシュボードの **「新しい動画を作成」** ボタンをクリックします
2. アップロード画面が表示されます

   ![アップロード画面のイメージ]

3. 音声ファイルまたはトランスクリプトファイルを画面中央の **点線のエリアにドラッグ&ドロップ** します
   - または **「ファイルを選択」** ボタンからファイルを選べます

4. 以下の情報を入力します:
   - **会議タイトル**（必須）: 例「2026-05-19 定例MTG」
   - **説明**（任意）: 会議の概要
   - **話者数**（任意）: 不明な場合は空白でOK（自動検出されます）

5. **「処理を開始する」** ボタンをクリックします

### 2.2 対応ファイル形式

| 形式 | 拡張子 |
|---|---|
| MP3 | .mp3 |
| WAV | .wav |
| MP4（音声付き動画） | .mp4 |
| M4A | .m4a |
| OGG | .ogg |
| WebM | .webm |
| Text | .txt |
| Markdown | .md |
| SRT | .srt |
| VTT | .vtt |

**最大ファイルサイズ**: 500MB  
**最大録音時間**: 3時間

---

## 3. 処理の進捗確認

アップロードが完了すると、自動的に処理が始まります。

### 3.1 処理ステップの説明

| ステップ | 内容 | 目安時間 |
|---|---|---|
| ① 入力アップロード | ファイルをサーバーに保存 | 1〜3分 |
| ② 文字起こし（音声入力時のみ） | AIが音声をテキストに変換 | 2〜10分 |
| ③ 話者分離（音声入力時のみ） | 誰が話しているか識別 | 1〜5分 |
| ④ AI分析 | 要約・テーマ・名言を生成 | 1〜2分 |
| ⑤ 字幕生成 | 字幕ファイルを作成 | 30秒 |
| ⑥ 動画生成 | アバター動画をレンダリング | 5〜15分 |
| ⑦ YouTube投稿 | 動画をYouTubeにアップロード | 2〜5分 |

**30分の音声の場合、合計で約20〜30分かかります。**

### 3.2 進捗画面の見方

処理状況はリアルタイムで更新されます:
- ✅ **緑のチェック**: 完了
- 🔄 **青いくるくる**: 処理中
- ⏳ **グレーの時計**: 待機中
- ❌ **赤のバツ**: エラー

---

## 4. 動画のプレビューと確認

処理が完了すると、通知が表示されます。

### 4.1 プレビュー確認手順

1. 通知をクリック、またはダッシュボードからジョブをクリックします
2. **「動画プレビュー」** タブをクリックします
3. 動画プレイヤーで内容を確認します

### 4.2 確認できる内容

**「コンテンツ」タブ**:
- 文字起こし全文
- AI生成の要約（3段階）
- 会話テーマ・キーワード
- 名言・印象的な発言
- 字幕ファイルのダウンロード

**「動画」タブ**:
- フル動画のプレビュー再生
- 動画ファイルのダウンロード

---

## 5. YouTubeへの投稿

### 5.1 投稿手順

1. **「YouTube投稿」** ボタンをクリックします
2. 投稿設定画面が表示されます

   **タイトル**: AIが自動生成したタイトル（編集可能）  
   **説明文**: AIが自動生成した説明（編集可能）  
   **タグ**: AIが自動生成したタグ（追加・削除可能）  
   **公開設定**:
   - 🔒 非公開（自分だけ見られる）
   - 🔗 限定公開（リンクを知っている人だけ）
   - 🌐 公開（全員が見られる）

3. 内容を確認・編集して **「投稿する」** ボタンをクリックします
4. 投稿完了後、YouTubeのURLが表示されます

### 5.2 投稿をやめたい場合

投稿をしない場合は **「スキップ」** ボタンをクリックしてください。  
動画ファイルのダウンロードリンクが表示されます。

## 6. よくある質問（FAQ）

### Q: 日本語以外の音声に対応していますか？
A: 英語にも対応しています。アップロード時に言語を選択してください。複数言語が混在する場合は自動検出されます。

### Q: 文字起こしの精度が低いのですが？
A: いくつかの原因が考えられます:
- 録音品質が悪い（雑音が多い、音量が低い）
- 複数人が同時に話している部分が多い
- 専門用語や固有名詞が多い

文字起こし結果はプレビュー画面から手動で修正することができます（Phase 2で実装予定）。

### Q: 処理が途中で止まってしまいました
A: ジョブ一覧から該当のジョブを開き、**「再実行」** ボタンをクリックしてください。エラーが発生したステップから再開します。

### Q: 動画の品質を上げることはできますか？
A: 現在のMVP版では1種類のテンプレートのみです。今後のアップデートで複数テンプレートから選択できるようになります。

### Q: ファイルはどのくらいの期間保存されますか？
A: 音声ファイル/トランスクリプトファイルは30日間、動画ファイルは7日間保存されます。それ以降はサーバーから削除されます（YouTubeに投稿済みの動画は影響ありません）。

### Q: 無料で使えますか？
A: 個人利用目的では無料枠の範囲で運用できます。目安として、月10時間程度の音声処理であれば無料枠に収まります。

### Q: Google Meetの音声を直接取り込めますか？
A: 現在はファイルアップロードのみ対応しています。Google Meetで録音した音声ファイル、または会議メモのトランスクリプトファイルをアップロードしてください。

## 7. Google Meet音声の録音方法

ConversationMovieはGoogle Meet専用の録音ツールではありません。以下のいずれかの方法で音声ファイルを準備してください。

### 方法1: Google Meet内蔵録音機能（Google Workspace必要）
1. ミーティング中に画面下の「...」をクリック
2. 「ミーティングを録画」を選択
3. ミーティング後、Googleドライブから録画ファイルをダウンロード

### 方法2: OBS Studio（無料・おすすめ）
1. [OBS Studio](https://obsproject.com/) をインストール
2. 「音声のみ録画」モードで設定
3. ミーティング中に録音開始
4. 終了後にmp3/wav形式でエクスポート

### 方法3: iPhone/Androidのボイスレコーダー
スピーカーの近くにスマートフォンを置いてボイスメモアプリで録音する（音質は低め）

## 8. トラブルシューティング

### 「文字起こしに失敗しました」と表示される

**原因と対処法**:
1. ファイル形式が対応していない → 対応形式（mp3, wav, mp4）に変換してください
2. ファイルが破損している → 別の方法で音声を確認してください
3. サーバーの一時的なエラー → 「再実行」ボタンをクリックしてください

### 「YouTube投稿に失敗しました」と表示される

**原因と対処法**:
1. YouTubeの連携が切れている → 設定ページから再連携してください
2. YouTubeのAPIクォータが上限に達した → 翌日に再試行してください
3. 動画ファイルが破損している → サポートに連絡してください

### アップロードが途中で止まる

- インターネット接続を確認してください
- ファイルサイズが500MBを超えていないか確認してください
- ブラウザを更新して再試行してください

## 9. アカウント管理

### 9.1 データの削除
ジョブ一覧から「削除」ボタンをクリックすると、関連するすべてのデータが削除されます（YouTubeに投稿済みの動画は削除されません）。

### 9.2 アカウントの削除
設定ページの「アカウントを削除する」からアカウントを削除できます。すべてのデータが削除されます。

---

## 10. プライバシーポリシー（簡易版）

- アップロードされた音声ファイル/トランスクリプトファイルは30日後に自動削除されます
- 音声データは文字起こし・AI分析処理にのみ使用します
- 第三者への音声データの提供は行いません
- Gemini APIへのテキスト送信はGoogleのプライバシーポリシーに準じます

**重要**: 会議の録音前に、必ず全参加者から録音の同意を得てください。
