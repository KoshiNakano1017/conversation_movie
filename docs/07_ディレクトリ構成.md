## 5. 主要設定ファイル

### .env（実運用フォーマット）

```bash
# App
APP_ENV=development
APP_SECRET_KEY=dev-secret-key-change-in-production
APP_PORT=8000
APP_DEBUG=true

# DB: Supabase PostgreSQL（直接接続）
# Supabase Dashboard → Settings → Database → Connection string → Direct タブ
DATABASE_URL=postgresql://postgres:<password>@db.<project-ref>.supabase.co:5432/postgres

# Supabase（Settings → API）
SUPABASE_URL=https://<project-ref>.supabase.co
SUPABASE_ANON_KEY=eyJ...
SUPABASE_SERVICE_ROLE_KEY=eyJ...

# Gemini API（未設定でも起動可。分析実行時にエラー）
GEMINI_API_KEY=AIza...

# YouTube（未設定でも起動可。将来機能）
YOUTUBE_CLIENT_ID=
YOUTUBE_CLIENT_SECRET=
YOUTUBE_REDIRECT_URI=http://localhost:8000/api/youtube/callback

# Redis: Upstash（rediss:// = TLS 必須。REST URL ではない）
REDIS_URL=rediss://default:<token>@<endpoint>.upstash.io:6379

# Whisper（将来用）
WHISPER_MODEL=medium
WHISPER_DEVICE=cpu
WHISPER_MODEL_DIR=C:/Users/<user>/Desktop/ConversationMovie/models

# ローカルストレージ（Windows パス例）
MEDIA_DIR=C:/Users/<user>/Desktop/ConversationMovie/media
MAX_UPLOAD_SIZE_MB=500

# Remotion
REMOTION_PROJECT_PATH=C:/Users/<user>/Desktop/ConversationMovie/remotion
NODE_PATH=node
```

### 起動コマンド（PowerShell・複数ターミナル）

```powershell
# ターミナル1: API
cd backend
.\.venv\Scripts\Activate.ps1
uvicorn app.main:app --reload --port 8000

# ターミナル2: ai キュー（Gemini 分析）
cd backend
.\.venv\Scripts\Activate.ps1
celery -A app.celery_app worker -Q ai --concurrency=1 -P solo --loglevel=info

# ターミナル3: video キュー（Remotion レンダリング）
cd backend
.\.venv\Scripts\Activate.ps1
celery -A app.celery_app worker -Q video --concurrency=2 -P solo --loglevel=info

# 任意: Flower 監視
celery -A app.celery_app flower --port=5555
```

> Windows では Celery のデフォルト prefork は動作しないため `-P solo`（または `-P threads`）を必ず指定する。

### マイグレーション

```powershell
cd backend
.\.venv\Scripts\Activate.ps1
alembic upgrade head                              # Supabase Postgres へ適用
alembic revision --autogenerate -m "<message>"    # 新規マイグレーション作成
```
